## P4 — Lead-gen growth engine
- **Auto-call guardrails** (`lib/callGuardrails.js`): DNC (`leads.do_not_call`) +
  pan-US safe calling window (12–18 ET) + daily cap (`crm_settings.leadgen_daily_call_cap`).
- **Follow-up sequencer** (`lib/outreachSequencer.js`): DB-driven cadence
  (`crm_settings.leadgen_sequence`) — A1→A2(+7d)→**read-gated call**(+8d, only if A2
  opened + not signed up)→A8(+14d)→A20 breakup(+21d). Stops on reply / opt-out /
  signup. Repoints the open-pixel per step. CRM **"Auto follow-up"** toggle.
- **Reply classification** (`lib/replyClassify.js`): unsubscribe → `do_not_email`+
  `do_not_call`; "no thanks" → declined (stage lost); interested → warm + (gated) call.
